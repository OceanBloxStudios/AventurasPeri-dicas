Aventuras na Tabela Periódica

OCEAN BLOX STUDIOS

Objetivo do Jogo

O objetivo do jogo é completar missões científicas e coletar elementos químicos para resolver problemas e desafios científicos, enquanto se aprende sobre a Tabela Periódica de forma divertida e interativa.

Componentes do Jogo

  • Tabuleiro de Aventura
  • Cartões de Elementos
  • Cartas de Missão
  • Peões: Figuras representando os jogadores.
  • Dados: Para movimentação no tabuleiro.
  • Marcadores de Recursos: Representações de recursos como energia, água, etc., que são usados para completar missões.
  • Tabela Periódica Impressa: Para consulta durante o jogo.

Preparação do Jogo

  1. Configurar o Tabuleiro: Colocar o tabuleiro de aventura em uma superfície plana.
  2. Distribuir os Cartões de Elementos: Colocar os cartões de elementos virados para baixo nas áreas designadas do tabuleiro.
  3. Distribuir os Cartões de Missão: Colocar as cartas de missão em um monte separado.
  4. Escolher os Peões: Cada jogador escolhe um peão e o coloca na posição inicial no tabuleiro.

Regras do Jogo

  • Início do Jogo: Cada jogador recebe uma carta de missão inicial e três marcadores de recursos. Os jogadores jogam o dado para determinar a ordem de jogo.
  • Movimentação: Em cada turno, o jogador lança o dado e move seu peão pelo tabuleiro de acordo com o número obtido.
  • Coleta de Elementos: Quando um jogador para em uma área designada para coleta de elementos, ele pode pegar um cartão de elemento daquela área.
  • Completação de Missões: Os jogadores usam os elementos coletados para completar as missões. Cada missão completa oferece recompensas, como pontos ou recursos adicionais.
  • Cartas de Evento: Em algumas áreas do tabuleiro, os jogadores podem puxar cartas de evento que trazem desafios ou benefícios inesperados.
  • Consulta à Tabela Periódica: Os jogadores podem consultar a Tabela Periódica impressa para verificar as propriedades dos elementos e planejar suas estratégias.

Sobre o Jogo

Aventuras na Tabela Periódica é um jogo educativo projetado para tornar o aprendizado sobre a Tabela Periódica divertido e interativo. Jogadores de todas as idades podem se beneficiar do conhecimento adquirido enquanto se divertem completando missões e coletando elementos químicos. Foi feito por Minela, Lucas, Matheus e Devid
